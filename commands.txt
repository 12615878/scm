ls– list of files 
mkdir “file name”  - create file 
rmdir “file name” – delete directory 
pwd – present working directory 
cd “file name” – to get inside the file 
vi test.txt – new text file in folder.  Press I to insert text(enable edi ng text file) 
:wq – save and quit 
Cat test.txt “file name” – to display content inside the file 
Vi test.txt “file name” – to edit file again
